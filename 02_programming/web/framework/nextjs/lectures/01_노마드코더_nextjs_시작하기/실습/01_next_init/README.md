# 노마드 코더 "nextjs 시작하기"강의 프로젝트

## 개요

노마드코더에서 제공하는 "nextjs 시작하기" 강의를 수강하며 실습한 프로젝트입니다.  
영화관련 정보 제공 웹사이트를 만드는 프로젝트입니다.
[강의 사이트](https://nomadcoders.co/nextjs-for-beginners/lobby)

## 스크립트

### 1. 패키지 설치

```bash
npm install
```

### 2. 프로젝트 실행

```bash
npm run dev
```

### 3. 프로젝트 빌드

```bash
npm run build
```

### 4. 프로젝트 시작

```bash
npm start
```
